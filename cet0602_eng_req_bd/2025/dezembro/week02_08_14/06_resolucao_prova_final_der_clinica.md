# Conquista
Resolução completa da prova final de Engenharia de Requisitos com elaboração de Diagrama Entidade-Relacionamento para sistema de clínica médica, incluindo fundamentação lógica e versões compactas do diagrama.

# Data: 14-12-25
# Tempo Gasto: 51 minutos

# Processo Concluído

1. **Resolução de Questões Objetivas**
   - Análise e resposta das 7 questões objetivas da prova
   - Aplicação de conceitos de técnicas de elicitação, casos de uso, persistência, LGPD e modelagem

2. **Desenvolvimento do Diagrama Entidade-Relacionamento**
   - Elaboração de DER completo para sistema de gestão de clínica médica
   - Identificação de 5 entidades principais: MÉDICO, PACIENTE, QUARTO, INTERNAÇÃO, ATENDIMENTO
   - Definição de atributos essenciais baseados nos requisitos do enunciado
   - Estabelecimento de relacionamentos com cardinalidades adequadas

3. **Geração de Código Mermaid**
   - Criação de código para visualização do DER em plataformas online
   - Desenvolvimento de versões compactas para impressão em formato A4
   - Adaptação do diagrama para caber em captura de tela única

4. **Fundamentação Lógica do Modelo**
   - Elaboração de texto dissertativo justificando decisões de modelagem
   - Explicação do raciocínio por trás da estruturação das entidades e relacionamentos
   - Demonstração de como o modelo atende aos requisitos de relatórios do sistema

5. **Otimização para Avaliação**
   - Criação de versões ultracurtas do diagrama mantendo legibilidade
   - Sugestões de configuração para visualização e impressão
   - Abreviações estratégicas para economizar espaço sem perder significado
